# project title: Helix 2.0 Application Testing

## Description
This involves testing the **Update** features of Entry, Event and figures in helix
**Test scenario**
1. Testing with required fields filled.
2. Testing with required fields not filled.
3. Testing with selecting future dates.

## How to use
1. Clone the repository:  git clone <repository-url>
2. Navigate to the project directory:  cd <project-folder-name>
3. From the terminal install all the dependencies: npm install
4. Run playwright test: npx playwright test

## Required Software
1. Node.js (version 16 or above)
2. Playwright testing framework
3. VS Code (optional, for editing and running tests)
